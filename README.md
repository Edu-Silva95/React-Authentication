In this project I explored and learned a lot about React Authentication. This project uses the same files as "React Routing 2.0", but with the Authentication implemented for Users. 
This project was built with and for the "React - The Complete Guide 2025 (incl. Next.js, Redux)" - Udemy Course.

The topics of the lectures for this project:

- Module Introduction
- How Authentication Works
- Project Setup & Route Setup
- Working with Query Parameters
- Implementing the Auth Action
- Validating User Input & Outputting Validation Errors
- Adding User Login
- Attaching Auth Tokens to Outgoing Requests
- Adding User Logout
- Updating the UI Based on Auth Status
- Important: loader()s must return null or any other value
- Adding Route Protection
- Adding Automatic Logout
- Managing the Token Expiration
- Module Resources
